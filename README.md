# Power-BI-Sales-Dashboard
This project is an interactive Sales Dashboard built in Power BI to analyze sales performance, profit trends, customer behavior, and regional performance. The dashboard helps businesses track key KPIs, identify top-performing products, and make data-driven decisions using visual analytics.
